# 📊 Superstore Sales Dashboard | Power BI

## 📌 Project Overview

An interactive **Power BI dashboard** built to analyze Superstore sales data and deliver actionable business insights. The dashboard provides a comprehensive view of sales, profit, customer segments, shipping performance, and regional trends through interactive visualizations and KPIs.

## 🎯 Key Features

- 📈 Sales & Profit KPIs
- 📅 Monthly Sales and Profit Trends
- 🗺️ Regional & State-wise Sales Analysis
- 👥 Customer Segment Analysis
- 💳 Payment Mode Analysis
- 🚚 Shipping Mode Analysis
- 📦 Product & Sub-Category Performance
- 🎛️ Interactive Filters and Slicers

## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel

## 🧹 Data Preparation

- Cleaned and transformed data using Power Query
- Removed duplicates and handled missing values
- Created calculated columns and DAX measures
- Optimized data model for reporting
